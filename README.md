<h1>🐦 Flappy Bird</h1>

A simple recreation of the classic Flappy Bird arcade game built using web technologies. The goal of the game is to control a bird and fly through pipes without hitting them. The game tests your timing, reflexes, and focus as the bird continuously moves forward.

The original Flappy Bird became a viral mobile game where players guide a bird through gaps between pipes, earning points for every successful pass.

<h2>🎮 Gameplay</h2>

Press Spacebar / Click to make the bird flap upward.

Gravity continuously pulls the bird downward.

Navigate through the gaps between pipes.

Each pipe passed increases your score.

Hitting a pipe or the ground ends the game.

<h2>✨ Features</h2>

Smooth bird movement and gravity physics

Randomly generated pipes

Real-time score tracking

Game over detection and restart option

Lightweight and beginner-friendly project

<h2>🚀 Installation</h2>

Clone the repository:
```Bash
git clone https://github.com/Ananya9999999/Flappy-Bird.git
```

Go to the project folder:
```Bash
cd Flappy-Bird
```
Open index.html in your browser to start the game.
